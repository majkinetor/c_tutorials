C Enum Quick Reference

Enumerated types for readable, type-safe constants

Basic Syntax

enum Day {
    MONDAY,      // 0
    TUESDAY,     // 1
    WEDNESDAY,   // 2
    THURSDAY,    // 3
    FRIDAY,      // 4
    SATURDAY,    // 5
    SUNDAY       // 6
};

enum Day today = WEDNESDAY;
Values start at 0 by default and increment by 1

Explicit Values

enum Status {
    ERROR = -1,
    SUCCESS = 0,
    PENDING = 1,
    COMPLETED = 2
};

enum Flags {
    FLAG_A = 1,      // 0x01
    FLAG_B = 2,      // 0x02
    FLAG_C = 4,      // 0x04
    FLAG_D = 8       // 0x08
};

Using typedef

typedef enum {
    RED,
    GREEN,
    BLUE,
    ALPHA
} Color;

// Cleaner usage without 'enum' keyword
Color background = BLUE;
typedef eliminates need to write enum repeatedly

Switch Statements

switch (today) {
    case MONDAY:
    case TUESDAY:
    case WEDNESDAY:
    case THURSDAY:
    case FRIDAY:
        printf("Weekday\n");
        break;
    case SATURDAY:
    case SUNDAY:
        printf("Weekend\n");
        break;
}

Partial Assignment

enum Values {
    FIRST = 10,
    SECOND,        // 11
    THIRD,         // 12
    RESET = 0,
    NEXT           // 1
};
Unspecified values continue from previous + 1

Bit Flags Pattern

enum Permissions {
    READ    = 1 << 0,  // 0x01
    WRITE   = 1 << 1,  // 0x02
    EXECUTE = 1 << 2   // 0x04
};

int perms = READ | WRITE;
if (perms & EXECUTE) {
    // Has execute permission
}

Key Points

  • Enums are integers under the hood
  • Improve code readability vs magic numbers
  • No type safety - can assign any int value
  • Scope: enumerators are in the enclosing scope
  • Cannot have duplicate names in same scope
  • Use sizeof(enum Type) = sizeof(int)

Common Patterns

Size Tracking

enum Color {
    RED,
    GREEN,
    BLUE,
    COLOR_COUNT  // = 3
};

String Mapping

const char* day_names[] = {
    "Monday", "Tuesday", "Wednesday",
    "Thursday", "Friday",
    "Saturday", "Sunday"
};

printf("%s\n", day_names[MONDAY]);
