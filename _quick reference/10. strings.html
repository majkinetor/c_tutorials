C String Operations Manual

String Basics

In C, strings are arrays of characters terminated by a null character (\0). String literals are enclosed in double quotes.

char str1[20] = "Hello";
char str2[] = "World";  // Size determined automatically
char *str3 = "Constant"; // Points to string literal

String Declaration and Initialization

char str[50];                    // Uninitialized array
char str[50] = "Text";          // Initialized with string
char str[50] = {'H','i','\0'}; // Character by character
char str[] = "Auto-sized";      // Compiler determines size

Essential String Functions

Include the string library: #include <string.h>

strlen() - String Length

size_t strlen(const char *str);

Returns the number of characters before \0, excluding the null terminator.

char text[] = "Hello";
int len = strlen(text);  // Returns 5

strcpy() - String Copy

char *strcpy(char *dest, const char *src);

Copies src to dest, including the null terminator. Destination must have sufficient space.

char dest[20];
strcpy(dest, "Hello");

strncpy() - Safe String Copy

char *strncpy(char *dest, const char *src, size_t n);

Copies up to n characters. Does not automatically null-terminate if src is longer than n.

char dest[20];
strncpy(dest, "Hello World", 5);
dest[5] = '\0';  // Manual null termination

strcat() - String Concatenation

char *strcat(char *dest, const char *src);

Appends src to the end of dest. Destination must have enough space.

char str[20] = "Hello";
strcat(str, " World");  // str becomes "Hello World"

strncat() - Safe Concatenation

char *strncat(char *dest, const char *src, size_t n);

Appends up to n characters from src to dest.

strcmp() - String Comparison

int strcmp(const char *str1, const char *str2);

Returns:

  • 0 if strings are equal
  • Negative if str1 < str2
  • Positive if str1 > str2
if (strcmp(str1, str2) == 0) {
    // Strings are equal
}

strncmp() - Compare N Characters

int strncmp(const char *str1, const char *str2, size_t n);

Compares only the first n characters.

strchr() - Find Character

char *strchr(const char *str, int c);

Returns pointer to first occurrence of character c, or NULL if not found.

char *pos = strchr("Hello", 'l');  // Points to first 'l'

strrchr() - Find Last Character

char *strrchr(const char *str, int c);

Returns pointer to last occurrence of character c.

strstr() - Find Substring

char *strstr(const char *haystack, const char *needle);

Returns pointer to first occurrence of substring, or NULL if not found.

char *pos = strstr("Hello World", "World");

Manual String Operations

Manual Copy

void my_strcpy(char *dest, const char *src) {
    int i = 0;
    while ((dest[i] = src[i]) != '\0') {
        i++;
    }
}

Manual Length

int my_strlen(const char *str) {
    int len = 0;
    while (str[len] != '\0') {
        len++;
    }
    return len;
}

Manual Concatenation

void my_strcat(char *dest, const char *src) {
    int i = 0, j = 0;
    while (dest[i] != '\0') i++;
    while ((dest[i++] = src[j++]) != '\0');
}

Input/Output Functions

Include: #include <stdio.h>

Reading Strings

char str[100];
scanf("%s", str);           // Reads until whitespace
fgets(str, 100, stdin);     // Reads entire line, includes '\n'
gets(str);                  // Unsafe, deprecated

Printing Strings

printf("%s\n", str);
puts(str);                  // Adds newline automatically

Common Pitfalls

  1. Buffer overflow: Ensure destination has sufficient space
  2. Missing null terminator: Always ensure strings are null-terminated
  3. Uninitialized arrays: Initialize before use
  4. Modifying string literals: String literals are read-only
