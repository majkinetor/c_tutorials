# C tutorials

## Setup C compiler and editor

1. Install Visual Studio Code: `winget install vscode`
   1. [Install C/C++ extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools)
1. Install CodeBlocks C IDE/compiler
   1. Via shell: `winget install CodeBlocks.CodeBlocks.MinGW`
   1. Manual: [codeblocks-25.03mingw-setup.exe](https://sourceforge.net/projects/codeblocks/files/Binaries/25.03/Windows/codeblocks-25.03mingw-setup.exe)
1. Make sure mingw `bin` directory is on the `PATH` so that vscode can find it
